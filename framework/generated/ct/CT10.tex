\subsection{CT10: Finite refinement classification}
\label{sec:ct10-automation}

CT10 records the class-table stage as a nullary audit boundary and searches the exact class universe for a direct case. Exhaustive direct absence is an explicit predecessor of the first-empty-row search, which evaluates rows from the original input. An empty row already contains its canonical promotion; the final promotion node records the terminal audit edge.

A theorem instantiation supplies the problem-specific static entries in the first table. The framework constructs every remaining node output, exhaustive decision, certificate, residual, edge, and trace theorem.

\paragraph{Public automation surface.}
\texttt{StructuralExhaustion.CT10.run} returns the evidence-carrying \texttt{ExecutionResult}. \texttt{ct10\_\allowbreak{}execute}, \texttt{ct10}, and \texttt{ct10\_\allowbreak{}total} expose execution, semantic soundness, and totality, respectively.
