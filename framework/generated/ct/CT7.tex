\subsection{CT7: Exact context classification}
\label{sec:ct7-automation}

CT7 separates realization from response comparison. The first search either produces a realizing context for the left object or an exhaustive unrealized state. Only that state enables the second search, which returns a concrete response distinction or a neutrality certificate for every declared context.

A theorem instantiation supplies the problem-specific static entries in the first table. The framework constructs every remaining node output, exhaustive decision, certificate, residual, edge, and trace theorem.

\paragraph{Public automation surface.}
\texttt{StructuralExhaustion.CT7.run} returns the evidence-carrying \texttt{ExecutionResult}. \texttt{ct7\_\allowbreak{}execute}, \texttt{ct7}, and \texttt{ct7\_\allowbreak{}total} expose execution, semantic soundness, and totality, respectively.
