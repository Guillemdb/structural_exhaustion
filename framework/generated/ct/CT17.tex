\subsection{CT17: Finite target thickening and survivor arithmetic}
\label{sec:ct17-automation}

CT17 first checks every target-offset pair for compatibility and then performs the literal finite-scale split. At finite scale it enumerates the exact survivor list, returning exhaustion or a nonempty survivor residual. At orbit scale it searches target-offset arithmetic, returning a target-hit certificate or an orbit-avoidance residual with the computed value list.

A theorem instantiation supplies the problem-specific static entries in the first table. The framework constructs every remaining node output, exhaustive decision, certificate, residual, edge, and trace theorem.

\paragraph{Public automation surface.}
\texttt{StructuralExhaustion.CT17.run} returns the evidence-carrying \texttt{ExecutionResult}. \texttt{ct17\_\allowbreak{}execute}, \texttt{ct17}, and \texttt{ct17\_\allowbreak{}total} expose execution, semantic soundness, and totality, respectively.
