\subsection{CT16: Exact whole-support closed type}
\label{sec:ct16-automation}

CT16 searches the exact coordinate enumeration for a point outside support. A missing coordinate gives the proper-support residual. Whole support enables closed-code computation, after which literal decidable equality with the target code yields either the exact-code certificate or the mismatch residual.

A theorem instantiation supplies the problem-specific static entries in the first table. The framework constructs every remaining node output, exhaustive decision, certificate, residual, edge, and trace theorem.

\paragraph{Public automation surface.}
\texttt{StructuralExhaustion.CT16.run} returns the evidence-carrying \texttt{ExecutionResult}. \texttt{ct16\_\allowbreak{}execute}, \texttt{ct16}, and \texttt{ct16\_\allowbreak{}total} expose execution, semantic soundness, and totality, respectively.
