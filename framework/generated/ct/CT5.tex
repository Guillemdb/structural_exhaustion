\subsection{CT5: Local witness aggregation}
\label{sec:ct5-automation}

CT5 searches for an active site with no supporting witness. Exhaustive absence produces the deficit-free predecessor used to compute the local contribution ledger. The fixed-priority arithmetic comparison returns C4 when capacity is below the required amount, an aggregate residual when the computed total exceeds capacity, and otherwise a bounded charge ledger.

A theorem instantiation supplies the problem-specific static entries in the first table. The framework constructs every remaining node output, exhaustive decision, certificate, residual, edge, and trace theorem.

\paragraph{Public automation surface.}
\texttt{StructuralExhaustion.CT5.run} returns the evidence-carrying \texttt{ExecutionResult}. \texttt{ct5\_\allowbreak{}execute}, \texttt{ct5}, and \texttt{ct5\_\allowbreak{}total} expose execution, semantic soundness, and totality, respectively.
