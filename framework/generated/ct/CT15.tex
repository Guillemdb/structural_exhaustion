\subsection{CT15: Target-relative rank and full-rank ledger}
\label{sec:ct15-automation}

CT15 computes target-relative rank from the invocation context and searches for the first target-dependent coordinate. A hit is the rank-drop residual; exhaustive absence proves full rank and enables the ordered charge ledger. The ledger-capacity comparison returns C4 or the reusable full-rank ledger residual.

A theorem instantiation supplies the problem-specific static entries in the first table. The framework constructs every remaining node output, exhaustive decision, certificate, residual, edge, and trace theorem.

\paragraph{Public automation surface.}
\texttt{StructuralExhaustion.CT15.run} returns the evidence-carrying \texttt{ExecutionResult}. \texttt{ct15\_\allowbreak{}execute}, \texttt{ct15}, and \texttt{ct15\_\allowbreak{}total} expose execution, semantic soundness, and totality, respectively.
