# API reference

The reference is generated from the compiled Hypostructure Lean environment. It includes the public framework declarations selected by the exporter plus native fixture and example endpoints linked to hash-verified source ranges.

## Modules

Module pages show layer ownership, direct imports, documentation, and declarations. Fixture and application modules remain visibly separate from reusable Core, CT, Routes, Graph, and PDE APIs.

## Declarations

Declaration pages show the fully qualified name, declaration kind, elaborated type, hash-verified source range, documentation or a structural explanation, direct dependency names, and whether a body is present. These are source facts; mathematical trust remains governed by Lean and explicit theorem boundaries.
