# Examples

Examples are compiled source packages that demonstrate framework behavior without moving semantic computation into the browser.

## Neutral fixtures

Core fixtures exercise residuals, decisions, focus, coordinates, assembly, finite search, budgets, routing, closure, and every CT outcome on small finite models.

## Graph examples

Graph fixtures cover finite objects, isomorphism, induced structure, deletion, progress, rooted returns, boundaried atoms, gluing, responses, and graph CT adapters.

## PDE examples

The PDE package supplies a finite scalar model, nested windows, coordinate paths, gauges, compact extraction, additive local-tail assembly, and a represented two-dimensional Navier–Stokes registration.

## Full application

The Erdős case study shows how a theorem-specific package remains thin: nodes consume literal predecessors, query inherited evidence, and call framework or Graph executors.
