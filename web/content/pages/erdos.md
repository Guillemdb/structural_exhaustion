# Erdős–Gyárfás case study

This application registers the graph problem, official target, initial residual, and theorem-specific algebra, then follows the complete 157-node proof structure recorded by the immutable paper.

## Reading the graph

Every paper node has a page. Directed edges come from the paper's direct-predecessor relation, including joins and later-numbered predecessors; numeric order is only a display convention.

## Evidence

The paper supplies node references, normalized inputs and outcomes, CT labels, and predecessor edges. Native Hypostructure modules supply source links, declarations, executors, and explicit axiom-audit commands as implementation evidence.

## Trust boundary

The application imports Hypostructure and its own earlier modules. Any external theorem dependency remains visible as a named declaration and explicit trust boundary.
