# Migration issues

This directory records mathematical or specification blockers discovered
while translating an immutable authority into Hypostructure. An issue is not
an approved correction and does not modify the authority.

For Erdős--Gyárfás Problem 64, each issue must quote its exact obligation from
`original_erdos_64_proof.tex`, distinguish kernel-checked facts from proposed
repairs, identify any stronger legacy premise, and list every blocked paper
node. A proposed additional hypothesis or changed definition requires explicit
approval before an EG node may instantiate it.

For PDE rows, an issue must name the exact notebook or TeX contract, separate
finite/algebraic fixture evidence from analytic integration, expose every
missing primitive or residual, and forbid promotion based only on a degenerate
toy instance.
