# Erdős 64 node repair templates

This directory records the mandatory identity card and obligation ledger for
each migrated Chapter 1 diagram node.  A template is written before Lean is
edited and remains the audit record for the exact paper-local responsibility,
incoming stage, ledger queries, framework CT/combinator, outgoing edges, and
kernel-check status.

Templates are processed in directed dependency order.  They do not change the
immutable topology in `original_erdos_64_proof.tex`.
