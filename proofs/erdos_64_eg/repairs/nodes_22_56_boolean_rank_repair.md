## Quantifier normalization

For a finite response system `value : State -> Coordinate -> Bool`, complete
realization is

```
forall assignment : Coordinate -> Bool,
  exists state, value state = assignment.
```

The following weaker statements are insufficient:

- every coordinate separately takes both values;
- every response-preserving quotient code is injective;
- every barrier separately has a safe connector;
- independently chosen connector paths exist for separate barriers.

`Core.LocalBooleanRealization.System.classify` selects the first such
