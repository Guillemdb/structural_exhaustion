# Implement a Target Avoidance Strategy

Take candidate schedules and local pieces from the literal predecessor. Validate explicit target witnesses or exhaust the exact registered schedule; never search an ambient universe.

Compose target realization, avoidance, local reduction, criticality, and inherited semantic continuation. Core owns the dichotomy, branch witness retention, and target/residual routing.

The target branch must carry a proof of the registered `Core.Target.Predicate`. Every complementary branch must carry the exact residual required by its next strategy. Prove exhaustive coverage and locality, then expose the result as a registered DAG vertex; Core owns output construction and finalization.

Read only the previous residual and accumulated ledger. Route missing public machinery through `extend-hypostructure-framework`.
